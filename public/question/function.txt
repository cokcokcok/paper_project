<h3>attack 함수와 spawn함수를 만들어보자</h3>
<h5>attack() : unit의 상태가 적과 접촉했을때에 attack를 출력하는 함수를 만드시오</h5>
<h5>spawn() : 반복문을 활용해 unit를 5번 출력하는 함수를 만드시오</h5>
<hr>

<h4>함수란?</h4>
<ul>
	<li>함수는 작은 프로그램의 단위 입니다.</li>
	<li>우리가 지금까지 사용해온 printf()역시 c언어에서 미리 만들어 제공한 함수입니다.</li>
	<li>함수의 구조는 반환형태 함수이름(매개변수이름)으로 이루어져 있습니다.</li>
	<li>함수의 형태는 아래와 같습니다.</li>
<div class="row" id="base_from" style="height:32vh">// 함수의 형태
#include &ltstdio.h&gt

int sum(int a, int b) {
	int c = a + b;
	return c;
}

int main() {
	int num1 = 3, num2 = 5;
	int total = sum(num1, num2);
	
	printf("%d\n", total);
	return 0;
}</div>
	<li>sum()을 메인 함수에서 사용하기 위해서는 메인함수 보다 위에 선언해야 합니다.</li>
	<li>sum 함수 설명</li>
	<ul>
		<li>반환형태 : 정수형 값을 반환하는 함수이다.</li>
		<li>함수이름 : sum이라는 이름을 가진 함수이다.</li>
		<li>매개변수 : 받은 값을 정수형 변수 a, b로 사용하겠다</li>
		<li>설    명 : 정수형 변수를 a, b로 받아 두수를 더한뒤 값을 반환 하겠다.</li>
	</ul>
	<li>반환할 값이 없다면 void로 지정해주면 된다.</li>
</ul>
