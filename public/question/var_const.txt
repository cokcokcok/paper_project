<h4>변수란?</h4>
<ul>
	<li>저장 되어있는 값이 바뀔수 있는 수!</li>
	<li>변수 선언시 주의 사항</li>
	<ul>
		<li>변수의 이름은 알파벳, 숫자, 언더바(_)로 구성된다.</li>
		<li>C언어는 대소문자를 구분한다.</li>
		<li>변수의 이름은 숫자로 시작할 수 없다</li>
		<li>이름 사이에 공백이 삽입될 수 없다.</li>
		<li>키워드를 변수의 이름을 사용할수 없다.</li>
	</ul>
	<li>C 언어에서의 형식</li>
	<ul>
		<li>int lv = 1;</li>
		<li>int : 정수를 표현 하는 자료형</li>
		<li>lv : 변수의 이름</li>
		<li>1 : 변수에 넣을 값</li>
	</ul>
</ul>
