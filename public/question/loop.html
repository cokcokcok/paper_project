반복문을 활용하여 "unit"을 다섯번 출력하는 코드를 만들어 보자.

출력시 ','단위로 구분하시오

반복문이란?

  • 특정 조건에 한해서 어떠한 일을 반복 시키는 문법이다.
  • 반복문에는 for문과 while문이 있다.
  • for문은 초기식, 조건식, 증감식이 묶여있는 문법이다.
  • for문의 형식은 아래와 같다.
    • // for문 for(int num = 0; // 초기식 (반복을 위한 변수선언) num < 3; // 조건식 (반복의 범위를 지정) num++) // 증감식 (반복의 범위를 이탈하기 위한 증가 혹은 감소) { // 반복이 되는 문장 printf("wa!"); } // 한줄로 작성시 // for(int num = 0; num < 3; num++)
  • 실행 결과 : wa!wa!wa!
